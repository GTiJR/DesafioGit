O QUE É GIT
Git é uma ferramente computacional que nos permite (programadores) compartilhar códigos entre si de uma forma prática, de tal modo que os programadores de um mesmo projeto podem trabalhar neste ao mesmo tempo e ao final mergir as partes alteradas do código. Pelo meu entendimento, o programa compara as versões anteriores do código com as que foram alteradas e assim atualiza a versão que fica na branch master, que é a versão "definitiva". O git é bem útil pois com um pouco de comunicação (na hora de baixar a versão da brach master), todos os envolvidos podem fazer seu trabalho nas suas branches locais e ao final, mergir todas para o que seria a versão "definitiva" atualizada. 

O QUE É GITHUB
Github é a plataforma que permite aos programadores armazenar os projetos git online, de tal forma que eles fiquem hospedos na plataforma e pronto para serem baixados da brach master que é online e alterados nas braches locais. É o local onde o projeto git com o código "definitivo" mensionado na definição anterior de git fica hospedado. Com o github também é possível restaurar versões anteriores do projeto, de tal modo que se algum dos programadores cometer um erro em seu código e não estiver conseguindo identificar o erro, ele pode recorrer ao github e fazer uma restauração do projeto para uma versão anterior deste.
