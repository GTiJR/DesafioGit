O QUE É GIT?
O GIT é um sistema que proporciona um controle e uma manipulação de diversos arquivos de forma mais organizada. Com ele é possivel trabalhar tranquilamente no desenvolvimento de algum projeto com o auxilio de varias pessoas, sem ter que ficar sempre olhando o que o outro modificou no arquivo original e sem o perigo de ter alguma alteração sobrescrita. Isso acontece em virtude da "branch", uma especie de copia do arquivo original que pode ser criada por qualquer participante do projeto e torna possivel o desenvolvimento individual para depois ser juntado em um produto final de forma harmonica, facilitando o trabalho em grupo de um projeto.
