O QUE É O GIT?
  GIT É UM SISTEMA DE CONTROLE DE VERSÃO QUE PERMITE A PARTICIPAÇÃO DE DIVERSOS DESENVOLVEDORES EM UM MESMO PROJETO. O GRANDE DIFERENCIAL É A EDIÇÃO DE UM MESMO CÓDIGO POR DIFERENTES PESSOAS, SEM FICAR CONFUSO E DESORGANIZADO.
 O QUE É O GITHUB?
  LOCAL PARA OS DESENVOLVEDORES HOSPEDAREM SEUS PROJETOS E POSSUI ALGUMAS FUNÇÕES DE REDES SOCIAIS, CONECTANDO PESSOAS DE DIVERSAS REGIÇÕES DO GLOBO, TODOS VISANDO À COOPERAÇÃO EM UM PROJETO.
